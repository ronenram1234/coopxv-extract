# Bug Fix Workflow

## Phase 1: Investigation (Ronen coordinates)
1. Create save point
2. Identify root cause

## Phase 2: Fix Planning (Ronen)
1. Determine approach
2. Assess risk
3. Get approval if high-risk

## Phase 3: Implementation
1. Implement fix
2. Add regression test

## Phase 4: Testing (Lisa)
1. Verify bug fixed
2. Check regressions

## Phase 5: Deploy (Michael)
1. Deploy staging
2. Deploy production
3. Monitor
